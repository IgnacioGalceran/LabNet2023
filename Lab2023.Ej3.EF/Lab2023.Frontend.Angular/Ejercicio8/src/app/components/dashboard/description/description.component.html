Aplicación Angular

Descripción del Ejercicio 8

Este ejercicio consiste en la creación de una aplicación SPA (Single Page Aplication) frontend usando Angular, la cual interactúa con tres diferentes API's:

  • Una API creada por mi en C# con .NET Framework (Proyecto Web API), que proporciona y manipula datos de la entidad 'Customer' de la base de datos Northwind de Microsoft.
  • Una API pública 'https://nominatim.openstreetmap.org', que se utiliza para localizar ciudades en base a su nombre o zona geográfica.
  • Una API pública '`https://api.weatherapi.com' que ofrece información meteorológica detallada basada en la localización proporcionada por la API anterior. Esta API no es gratis, tiene un plan free de 14 dias, por lo que el 27 de Septiembre este servicio quedará en desuso (habría que crearse otra cuenta en la misma plataforma y generar otra KEY).

La aplicación Angular se encarga de solicitar datos a estas APIs, mostrar la información al usuario y ofrecer una experiencia de usuario fluida y amigable.

La API de C# permite realizar operaciones de alta, baja y modificación (ABM) en datos de clientes, lo que proporciona funcionalidad básica de administración de clientes

Este ejercicio combina la creación de componentes frontend con la integración de servicios web externos y la administración de diferentes datos.
