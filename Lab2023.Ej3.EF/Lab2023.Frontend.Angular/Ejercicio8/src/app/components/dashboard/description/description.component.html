Aplicación Angular

Descripción del Ejercicio 8

Este ejercicio consiste en la creación de una aplicación frontend en Angular, la cual interactúa con tres APIs diferentes:

  • Una API creada en C# con .NET Framework (Web API), que proporciona datos personalizados y específicos de la aplicación. Esta API se utiliza para obtener y manipular datos de clientes.
  • Una API pública 'https://nominatim.openstreetmap.org', que se utiliza para localizar ciudades en base a su nombre o zona geográfica.
  • Una API pública '`https://api.weatherapi.com' que ofrece información meteorológica detallada basada en la localización proporcionada por la API anterior. Esta API no es gratis, tiene un plan free de 14 dias, por lo que el 29 de Septiembre este servicio quedará en desuso.

La aplicación Angular se encarga de solicitar datos a estas APIs, mostrar la información al usuario y ofrecer una experiencia de usuario fluida y amigable.

La API de C# permite realizar operaciones de alta, baja y modificación (ABM) en datos de clientes, lo que proporciona funcionalidad básica de administración de clientes

Este ejercicio combina la creación de componentes frontend con la integración de servicios web externos y la administración de datos de clientes.
